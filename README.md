# AutoTranslate

Простой Telegram бот для автоматического перевода текста между раскладками QWERTY и ЙЦУКЕН.

## Зачем это нужно

Этот бот был создан для автоматического перевода текста, набранного на неправильной раскладке клавиатуры в Telegram. Особенно полезен в чатах, где участники часто забывают переключить раскладку.

## Требования

- Python 3.7+
- telethon

## Установка

1. Установите зависимости:
   ```
   pip install telethon
   ```

2. Склонируйте репозиторий:
   ```
   git clone https://github.com/VoidEmptiness/AutoTranslate.git
   cd AutoTranslate
   ```

3. Настройте конфигурацию в `AutoTranslate.py`:
   - Замените `api_id` и `api_hash` на свои (получить можно в https://my.telegram.org)
   - Укажите нужный `TARGET_CHAT` ID (Можно получить через @WhatChatIDBot)

## Запуск

```
python AutoTranslate.py
```

После первого запуска бот запросит номер телефона для авторизации.

### Keyboard Layout
Раскладка настроена в словаре `layout` и соответствует стандартной раскладке QWERTY → ЙЦУКЕН
