# AutoTranslate

Telegram бот для автоматического перевода текста между раскладками QWERTY и ЙЦУКЕН с интеллектуальным распознаванием английских слов.

## Примеры

**Было:** `Рфккн Зщееук фтв еру Ыщксукукэы Ыещту` → **Стало:** `Harry Potter and the Sorcerer's Stone`

## Требования

- Python 3.7+
- telethon
- requests

## Установка

1. Установите зависимости:
   ```
   pip install telethon requests
   ```

2. Склонируйте репозиторий:
   ```
   git clone https://github.com/VoidEmptiness/AutoTranslate.git
   cd AutoTranslate
   ```

3. Настройте конфигурацию в `AutoTranslate.py`:
   - Замените `api_id` и `api_hash` на свои (получить можно в https://my.telegram.org)
   - Укажите нужный `TARGET_CHAT` ID (можно получить через @WhatChatIDBot)

## Запуск

```
python AutoTranslate.py
```

После первого запуска бот запросит номер телефона для авторизации.

## Конфигурация

### Раскладка клавиатуры
Раскладка настроена в словаре `layout` и соответствует стандартной раскладке QWERTY → ЙЦУКЕН

### API словаря
Бот использует бесплатный API словаря (https://api.dictionaryapi.dev) для распознавания английских слов. При недоступности API используется резервная локальная проверка, которая может работать неидеально и иногда неправильно определять английские слова.
